Make certain vanilla functional blocks instantly breakable by shift + right click when holding wrenches, just like these modded machines!
![Demonstration of wrenching blocks](https://raw.githubusercontent.com/ChloePrime/wrenchable-vanilla/refs/heads/master/promotional_materials/wrenching_blocks.avif) 
Certain entity-based functional blocks are also supported!
![Demonstration of wrenching entity based functional blocks](https://raw.githubusercontent.com/ChloePrime/wrenchable-vanilla/refs/heads/master/promotional_materials/wrenching_entities.avif)
Note: This mod does not provide wrenches by itself, you need to install any tech mod that has wrenches, or add wrench tags to existing items to make this mod actually useable.
